# react-webpack

### Supports
  
  - Babel
  - Sass
  - React router
  
#
### Installation
  
  ```
    git clone https://github.com/CanKolay7363/react-webpack.git
    
    cd react-webpack
    
    npm i
    
    npm i -g webpack webpack-cli onchange
  ```
  
#
### Dev server
  
  ```
    npm run dev
  ```
  
  and open `public/index.html` on browser
